# ClientHub Frontend

Production-style React frontend for ClientHub, a CRM application designed to help users manage clients, track activities, and support day-to-day relationship workflows through a clean and scalable user interface.

---

## Overview

This project is the frontend application for the ClientHub ecosystem.

It is being built to demonstrate:

- Modern React frontend architecture
- Reusable UI components
- Structured routing and layouts
- API integration with a Spring Boot backend
- Scalable state management patterns
- Production-minded project organization

The application will allow authenticated users to:

- View and manage clients
- Track client activities
- Navigate dashboard-style CRM workflows
- Interact with secure backend APIs
